# What is SQL?
SQL (Structured Query Language) is a programming language used to store, manage, and retrieve data in relational databases.
SQL is the language you use to talk to databases. There are many database use SQL: 
MySQL
Microsoft SQL Server
Oracle
Simple syntax can allow to 
- create and modify databases
- insert data
- query data
- update data
- delete data and so on
